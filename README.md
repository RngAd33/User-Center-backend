# 用户中心项目

## 基于Ant Design Pro和Spring Boot的一站式用户管理系统
## 可以独立运行，也可以嵌入到其他项目当中作为后台管理模块，方便用户管理、提供个性化服务

### Q：这个项目采用了哪些技术选型？
### A：- 前端部分：
### Ant Design Pro：基于Ant Design和React的快速开发框架
### UmiJS：基于React的快速开发框架
### Umi Request 请求库

### - 后端部分：
### Spring Boot 后端框架
### MyBatis + MyBatis Plus 框架
### JUnit 单元测试框架
